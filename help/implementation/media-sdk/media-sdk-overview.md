# Media SDK概述 {#implementation-overview}

将流媒体SDK实施到您的视频播放器框架和视频应用程序中，以收集Web、移动设备和OTT平台的流媒体数据。  通过我们同类最佳的媒体SDK和API收集，可以轻松共享、分段和审核您收集的关键量度。 深入研究 — 将流媒体量度与其他Adobe Analytics解决方案和数据点（如受众分析或移动设备应用程序访问和人员量度）相结合。
