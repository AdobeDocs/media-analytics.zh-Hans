# Media SDK 概述 {#implementation-overview}

在视频播放器框架和视频应用程序中实施流媒体 SDK，以收集 Web、移动和 OTT 平台的流媒体数据。可通过我们同类最佳的 Media SDK 和 API 收藏集轻松共享、分段和审查您收集的关键量度。深入研究 – 将流媒体量度与其他 Adobe Analytics 解决方案和数据点相结合，例如受众分析或移动应用程序访问和人员量度。
