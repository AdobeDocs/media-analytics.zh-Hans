# 默认报表概述{#default-reports-overview}

Adobe Analytics 提供多种报告和量度，用于跟踪您网站上的媒体性能。

除了在启用每个模块时可用的量度和维度之外，在启用[!UICONTROL 媒体核心]模块时还有三个其他功能板样式报表可用。 启用[!UICONTROL 广告]模块还可以通过添加其他量度和过滤器来更改某些功能板样式报表的外观。

媒体报表列在&#x200B;**[!UICONTROL 报表]**&#x200B;选项卡上。 可导航至&#x200B;**[!UICONTROL 媒体]** > **[!UICONTROL 媒体参与度]**，然后从以下内容中选择：

| 媒体报表 | 描述 | 常见商业洞察 |
| --- | --- | --- |
| [媒体概述](media-reports-overview.md) | 显示多个聚合度量，以快速监视媒体是否正在按预期方式执行。 图表在广告展示次数旁显示[[!UICONTROL 媒体开始次数]](/help/reporting/metrics/media-starts.md)，以便您快速查看和比较这些量度。 | <ul> <li>热门量度的总数，包括[[!UICONTROL 内容开始]](/help/reporting/metrics/content-starts.md)、完成率、流平均时间和每次访问的平均媒体项目数。  </li> <li>按设备类型或国家/地区过滤的媒体的[[!UICONTROL 内容开始次数]](/help/reporting/metrics/content-starts.md)和[[!UICONTROL 广告开始次数]](/help/reporting/metrics/ad-starts.md)总计。  </li> </ul> |
| [媒体详细信息](media-reports-detail.md) | 显示所有媒体项目的详细量度，包括开始、并行查看者、完成率、播放百分比和广告展示次数。 | <ul> <li>热门量度的总数，包括[[!UICONTROL 媒体开始]](/help/reporting/metrics/media-starts.md)、[[!UICONTROL 内容开始]](/help/reporting/metrics/content-starts.md)和[[!UICONTROL 广告开始]](/help/reporting/metrics/ad-starts.md)以及每次访问的平均内容。  </li> <li>按设备类型或国家/地区过滤的媒体的[[!UICONTROL 内容开始次数]](/help/reporting/metrics/content-starts.md)和[[!UICONTROL 广告开始次数]](/help/reporting/metrics/ad-starts.md)总计。  </li> </ul> |
| [媒体时段](media-reports-daypart.md) | 显示[[!UICONTROL 到当天某个时间之前开始的内容]](/help/reporting/metrics/content-starts.md)，以便您能够在受众参与时快速进行查看。 | <ul> <li>按时间划分的受众参与。  </li> <li>将受众参与与以前的日期范围相比。  </li> </ul> |
| [媒体并行观众](media-concurrent-viewers-reports.md) | 显示一天内并行查看者。 数据不能按内容、设备类型或国家/地区进行过滤。 | <ul> <li>在 24 小时间隔内的每分钟受众参与。  </li> </ul> |
| 媒体事件和媒体变量 | 还提供其他报告。 媒体量度和维度是可以直接报告给其他 Analytics 报表并添加到其中的标准 Analytics 变量。 | <ul> <li>通过生成具有包含媒体内容类型的访问的报告来实现媒体转化（查看媒体后发生的事件）。  </li> <li>使用媒体名称 prop 的下一个/上一个媒体流。  </li> </ul> |
