# 调试仪表板和报告{#debug-dashboards-and-reports}

Adobe Debug提供实时报表，以便您查看在视频播放期间发送的点击和元数据。 其中每个报表都可以保存在Debug中。

为了认证，这些报告可以通过简单地复制URL并发送链接（例如，在ZenDesk票证中）来共享。

>[!NOTE]
>
>一次只能有一个会话处于活动状态。活动的会话可从功能板中打开。
