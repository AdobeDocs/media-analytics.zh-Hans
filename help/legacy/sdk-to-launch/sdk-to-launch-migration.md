# 从独立 Media SDK 迁移到 Adobe Launch

>[!NOTE]
>Adobe Experience Platform Launch 已更名为 Experience Platform 中的一套数据收集技术。 因此，产品文档中的术语有一些改动。 有关术语更改的综合参考，请参阅以下[文档](https://experienceleague.adobe.com/docs/experience-platform/tags/term-updates.html?lang=zh-Hans)。

本文档主要介绍Media Analytics实施差异
在独立Media SDK和Media Analytics Launch扩展之间。

Launch为客户提供了一种简单的方式来部署和管理所有分析，
营销和广告解决方案，为相关客户提供支持
跨Web和移动设备的体验。 Launch可让任何人构建和维护
他们自己的与Launch的集成。 这些集成称为扩展。
Launch客户可在应用商店中获取这些扩展，因此他们
可以快速安装、配置和部署其解决方案。

Adobe Media Analytics for Audio and Video 可用作 Web 和移动的 Launch 扩展：

* [Web Launch文档](https://experienceleague.adobe.com/docs/experience-platform/tags/extensions/adobe/media-analytics/overview.html)
* [Mobile Launch文档](https://developer.adobe.com/client-sdks/documentation/adobe-media-analytics/)
