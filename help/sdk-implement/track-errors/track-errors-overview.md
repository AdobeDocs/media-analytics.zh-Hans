## 实施错误跟踪

1. 跟踪媒体播放器错误。

   在发生错误事件时，使用错误信息调用 `trackError`。

>[!NOTE]
>
>跟踪媒体播放器错误不会停止媒体跟踪会话。如果媒体播放器错误导致无法继续播放，请确保通过调用 `trackSessionEnd` 后调用 `trackError` 来关闭媒体跟踪会话。
