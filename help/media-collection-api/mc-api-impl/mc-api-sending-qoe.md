# 发送 QoE 数据{#sending-qoe-data}

每个事件都可以使用名为 `qoeData` 的额外 JSON 键进行修饰，该键位于 JSON 请求正文中的 `params` 键旁边。

>[!NOTE]
>
>您应该查看 [JSON 验证架构](/help/media-collection-api/mc-api-impl/mc-api-validate-reqs.md)以验证参数类型以及它们是必选参数还是可选参数。
